# Diapositivas — Lección 1

Fuente Marp de la presentación:

```text
slides/leccion-01/leccion-01.md
```

Tema visual:

```text
slides/leccion-01/theme.css
```

Imágenes:

```text
slides/leccion-01/images/
```

## Exportar

Si tienes Marp CLI:

```powershell
npx @marp-team/marp-cli slides/leccion-01/leccion-01.md --theme slides/leccion-01/theme.css --pdf --output slides/leccion-01/dist/leccion-01.pdf
```

También puedes exportar a HTML:

```powershell
npx @marp-team/marp-cli slides/leccion-01/leccion-01.md --theme slides/leccion-01/theme.css --html --output slides/leccion-01/dist/leccion-01.html
```
