# Criterios de evaluación de voz

Usa esta checklist después de generar cada muestra de audio.

## 1. Naturalidad

Puntuación: `0–5`

- 0: robótica o artificial.
- 3: aceptable, pero se nota sintética.
- 5: natural y agradable para escuchar varios minutos.
