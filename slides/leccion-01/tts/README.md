# Prueba de voz sintética — Lección 1

Esta carpeta sirve para probar una voz sintética española antes de producir el videotutorial completo.

## Objetivo

Validar si la voz sirve para un curso técnico monetizable:

- suena natural;
- mantiene ritmo pausado;
- pronuncia bien términos técnicos;
- no cansa;
- transmite cercanía y autoridad.

## Archivos

- `texto-prueba-01.txt`: texto corto para la primera prueba de voz.
- `criterios-evaluacion.md`: checklist para decidir si una voz sirve.
- `prompt-voz.md`: descripción de la voz deseada.
- `audio/`: carpeta prevista para guardar audios exportados.
- `evaluaciones/`: fichas de evaluación por voz probada.

## Flujo recomendado

1. Copiar el contenido de `texto-prueba-01.txt`.
2. Abrir Google AI Studio o la herramienta TTS elegida.
3. Elegir una voz española.
4. Generar audio.
5. Guardar el resultado en `audio/`.
6. Evaluar con `criterios-evaluacion.md`.

## Nombres sugeridos para audios

```text
audio/google-ai-studio-voz-01.mp3
audio/google-ai-studio-voz-02.mp3
audio/elevenlabs-voz-01.mp3
```

No subas audios definitivos al repositorio si son muy pesados o si contienen restricciones de licencia. Para pruebas pequeñas sí puede ser útil guardarlos localmente.
