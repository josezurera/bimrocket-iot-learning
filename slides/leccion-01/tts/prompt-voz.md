# Prompt o descripción de voz deseada

Usar como referencia al elegir o generar una voz sintética.

```text
Voz española de España, natural, pausada y didáctica.
Estilo profesor técnico profesional, cercano y claro.
Debe sonar segura, pero no corporativa ni fría.
Ritmo tranquilo, con pausas naturales entre ideas.
Adecuada para un curso online de BIM, IoT y digital twins.
```

## Pronunciaciones a vigilar

La voz debe pronunciar de forma aceptable:

- BIMROCKET;
- IoT;
- API REST;
- JSON;
- BIM;
- CO₂;
- gemelo digital.

## Ritmo recomendado

Si la herramienta permite ajustar velocidad:

```text
Velocidad: 0.90–0.95
Estabilidad: media-alta
Expresividad: media
```

La prioridad es que el alumno entienda, no que parezca un anuncio.
