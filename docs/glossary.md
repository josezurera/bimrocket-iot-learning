# Glosario

**API REST**
: Interfaz HTTP mediante la cual una aplicación consulta o modifica recursos.

**BACnet**
: Protocolo habitual en automatización y control de edificios.

**BIM**
: Modelado de información de la construcción: geometría, propiedades y
  relaciones digitales de los activos construidos.

**Controlador**
: Componente de BIMROCKET asociado a un objeto que observa o provoca cambios.

**Fórmula**
: Expresión que mantiene una propiedad relacionada con otras propiedades del
  objeto.

**Gateway**
: Puente entre protocolos o redes de dispositivos y las aplicaciones que usan
  sus datos.

**Gemelo digital**
: Representación digital conectada con el estado y comportamiento de un activo
  real.

**IFC**
: Estándar abierto para intercambiar información BIM.

**IoT**
: Red de dispositivos capaces de medir, comunicar o actuar sobre el entorno.

**JSON**
: Formato textual estructurado muy utilizado por APIs web.

**MQTT**
: Protocolo ligero de publicación y suscripción utilizado frecuentemente en IoT.
