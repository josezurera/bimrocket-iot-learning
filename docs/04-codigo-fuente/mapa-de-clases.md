# Mapa de clases relacionadas con IoT

| Necesidad | Clase principal |
|---|---|
| Ciclo de vida de un controlador | `Controller` |
| Consultar una API periódicamente | `RestPollController` |
| Observar valores de Brain4it | `Brain4itWatchController` |
| Enviar órdenes a Brain4it | `Brain4itPostController` |
| Transformar dependencias | `Formula` |
| Representar un número mediante color | `ColorController` |
| Mostrar un valor y sus unidades | `DisplayController` |
| Mover un elemento | `TranslationController` |
| Girar un elemento | `RotationController` |
| Ejecutar lógica personalizada | `ScriptController` |
