# Itinerario de aprendizaje

## Fase 1 — Fundamentos

1. Seguir un dato desde un sensor hasta el modelo 3D.
2. Comprender JSON, HTTP y una API REST.
3. Distinguir sensor, gateway, plataforma IoT y gemelo digital.

## Fase 2 — BIMROCKET

4. Comprender objetos, eventos y controladores.
5. Conectar controladores mediante fórmulas.
6. Mostrar un valor y representar su estado mediante color.

## Fase 3 — Laboratorios

7. Crear un sensor REST simulado.
8. Consultarlo con `RestPollController`.
9. Vincular su valor a un objeto IFC.

## Fase 4 — Comunicación bidireccional

10. Recibir cambios mediante `Brain4itWatchController`.
11. Enviar órdenes mediante `Brain4itPostController`.
12. Construir un caso de iluminación o climatización.

## Fase 5 — Integración real

13. Incorporar MQTT o BACnet mediante un gateway.
14. Diseñar identificación, históricos y alarmas.
15. Añadir autenticación, reconexión y controles de seguridad.
